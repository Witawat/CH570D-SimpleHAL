# I2C

## Overview

I2C interface รองรับทั้ง master และ slave, 7-bit และ 10-bit addressing, 100kHz (standard) และ 400kHz (fast mode), และ SMBus

## SimpleHAL API

| ฟังก์ชัน | คำอธิบาย |
|---|---|
| `hal_i2c_init(cfg)` | เริ่มต้น I2C ตาม config struct |
| `hal_i2c_write(addr, data, len)` | เขียนไปยัง slave (master) |
| `hal_i2c_read(addr, buf, len)` | อ่านจาก slave (master) |
| `hal_i2c_write_raw(addr, data, len)` | เขียนแบบแยก transaction (master) |
| `hal_i2c_read_raw(addr, buf, len)` | อ่านแบบแยก transaction (master) |
| `hal_i2c_mem_write(dev_addr, mem_addr, mem_addr_size, data, len)` | เขียน register/memory ของ slave |
| `hal_i2c_mem_read(dev_addr, mem_addr, mem_addr_size, buf, len)` | อ่าน register/memory ของ slave |
| `hal_i2c_set_speed(hz)` | เปลี่ยนความเร็ว |

**Config struct (`hal_i2c_config_t`):** mode (`I2C` / `SMBUS_DEVICE` / `SMBUS_HOST`), own_addr (`uint16_t`), speed (`uint32_t` Hz), sda_pin, scl_pin

**Note:** `hal_i2c_write()` ใช้ combined transaction (repeated start) — ส่ง reg address + data ในครั้งเดียว เช่น sensor I²C ทั่วไป

## Circuit Guide

```
I2C Master (CH57x)      อุปกรณ์ I²C
  SDA (PA9) ───┬─────── SDA
               R(4.7k)
               VCC_3.3V
  SCL (PA10) ──┬─────── SCL
               R(4.7k)
               VCC_3.3V
  GND ──────── GND
```

| สาย | ขา CH57x | รายละเอียด |
|---|---|---|
| SDA | PA9 (default) | ต้องมี pull-up resistor 1k–10kΩ (4.7kΩ สำหรับ 100kHz, 2.2kΩ สำหรับ 400kHz) |
| SCL | PA10 (default) | ต้องมี pull-up resistor ค่าเดียวกับ SDA |

**Pull-up resistor selection:**

| Bus capacitance | 100kHz | 400kHz |
|---|---|---|
| ~50pF | 10kΩ | 4.7kΩ |
| ~100pF | 4.7kΩ | 2.2kΩ |
| ~200pF | 2.2kΩ | 1kΩ |

**Pin remap options:** SDA (PA9 / PA12), SCL (PA10 / PA13) — ใช้ `GPIOPinRemap(REMAP_I2C_MODE1)` สำหรับ PA12/PA13

## CH57x Specifics

- **Master + Slave:** รองรับทั้งสองโหมด — slave address ตั้งผ่าน config
- **7-bit และ 10-bit addressing:** เลือกตาม `own_addr` (0–127 = 7-bit, >= 128 = 10-bit)
- **DMA:** รองรับ
- **ฟิลเตอร์:** มี glitch filter บน SDA/SCL
- **SMBus:** รองรับทั้ง SMBus Device และ Host mode (timeout, alert)

## Limitations

| ข้อจำกัด | รายละเอียด |
|---|---|
| `hal_i2c_write_raw/read_raw` ต่างจาก `write/read` | `write/read` = combined transaction (repeated start), `_raw` = แยก transaction มี STOP คั่น — ใช้ผิดอาจทำให้ sensor ไม่ตอบสนอง |
| `mem_addr_size` | รองรับ 8-bit (`I2C_MEM_ADDR_SIZE_8BIT`) และ 16-bit (`I2C_MEM_ADDR_SIZE_16BIT`) |
| max speed | 400kHz (fast mode) — ไม่ support fast+ (1MHz) |
| pin remap จำกัด | SDA/SCL remap ได้แค่ PA12/PA13 เท่านั้น (นอกเหนือจาก default PA9/PA10) |
| ต่ออุปกรณ์หลายตัว | ต้องระวัง address ซ้ำ — scan ด้วย I2CScan library |

## Code สั้น

```c
hal_i2c_config_t i2c_cfg = {
    .mode = I2C_MODE_I2C,
    .speed = 100000,          // 100kHz
    .sda_pin = PA9,
    .scl_pin = PA10
};
hal_i2c_init(&i2c_cfg);

// เขียน register 0x10 ของ sensor (7-bit addr 0x68)
uint8_t data[] = {0x10, 0x00}; // reg addr + value
hal_i2c_write(0x68, data, 2);

// หรือใช้ mem_write สะดวกกว่า
uint8_t val = 0x00;
hal_i2c_mem_write(0x68, 0x10, I2C_MEM_ADDR_SIZE_8BIT, &val, 1);
```
